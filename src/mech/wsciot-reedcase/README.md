# Water resistant case for the reed sensor


This construction is written in Python using distance field based implicit surfaces.

To create the STL file you need to install Python and Poetry. After Installing these tools, run in command line (you must be in the directoriy, where this README.md is):

```bash
poetry install
poetry run python3 wsciot_reedcase
```

The generated files `reedcase.stl` and `reedcase.obj` will be placed in `gen/mech/wsciot-reedcase/`.
