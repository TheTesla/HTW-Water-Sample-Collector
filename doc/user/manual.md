# User Manual water sample collector IoT sensor

The intended use of the IoT sensor is to notify the user in the case, the water sample collect is full. It is extendable to enable data logging of additional sensors like conductivity and water level.

## Requirements

* Power source: 3.6 V Li-SOCl<sub>2</sub> AA form factor primary cell
* LoRa network coverage (868 MHz)
* Internet connected device with browser to check the state of the water sample collector online
